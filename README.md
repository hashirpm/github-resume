# GITHUB RESUME
An instant resume generator using Github.

## Demo
[Live Demo](https://gresume.herokuapp.com/)

## Technologies used
1. Github API
2. NodeJS
3. ExpressJS
4. Handlebars

## How to Run
**Step 1:**

Star the repository


**Step 2:**

Download or clone this repo by using the link below:

```
https://github.com/hashirpm/github-resume.git
```

**Step 3:**

Install Dependencies

```
npm install
```

**Step 4:**

Run the application

```
npm run start
```
